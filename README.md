# AI Chatbot Platform

A minimal Chatbot Platform built with FastAPI, PostgreSQL, and OpenAI integration.

## Overview

This platform allows users to:
- Create and manage AI chatbot projects/agents
- Configure custom system prompts
- Chat with AI agents using OpenAI or OpenRouter
- Upload files for context (OpenAI Files API)
- Manage multiple projects with secure authentication

## Tech Stack

- **Backend**: FastAPI (Python 3.11+)
- **Database**: PostgreSQL
- **Authentication**: JWT
- **LLM Integration**: OpenAI / OpenRouter
- **Deployment**: Railway
